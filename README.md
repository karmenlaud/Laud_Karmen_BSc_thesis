This is the code base for my thesis. 

**How to run the code:**
* Text detection is done using the openCV East module, instructions for use are found here https://github.com/gifflet/opencv-text-detection. The file frozen_east_text_detection.pb should be downloaded and located in the same repository with the rest of the codebase.
* All the modules in the requirements.txt file should be installed.
* For running the code, main.py should be run.
